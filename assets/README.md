# Assets

These are the visual files for the Stardance hardware submission.

- `system-diagram.png` / `system-diagram.md` - shows the robot data flow
- `wiring-diagram.png` / `wiring-diagram.md` - shows power, signal, ESP32-S3, DRV8833, encoder, and level-shifter wiring
- `chassis-sketch.png` / `chassis-sketch.md` - shows the physical robot layout
- `demo-plan.md` - lists the testing steps for the final demo video
