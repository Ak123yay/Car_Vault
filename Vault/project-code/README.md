# Project Code

This directory is reserved for future robot implementation.

Do not add production robot code until `TASKS.md` allows it and the relevant source documents in `wiki/_code-map.md` exist.

Current status:

- Phase 1 hardware purchase, bring-up, and measurement recording are still pending.
- Measured values must be recorded in `wiki/architecture/phase-1-measurement-worksheet.md` before calibration or control code is treated as ready.
- Arduino firmware will belong under `project-code/firmware/arduino_uno/` when implementation is allowed.
- Pi-side robot modules will belong under `project-code/car_vault/` when implementation is allowed.
