# PCB

This project does not need a custom PCB for this submission.

The robot uses ready-made boards:

- ESP32-S3 development board
- DRV8833 motor driver module
- 4-channel logic-level shifter
- Raspberry Pi 5

I included this folder because some Stardance/Outpost examples have a `pcb/` folder. For Car Vault, the wiring diagram explains the wiring instead of a custom PCB.
