# PCB

This project does not need a custom PCB for this submission.

The robot uses ready-made modules:

- ESP32-S3 development board
- DRV8833 motor driver module
- 4-channel bidirectional logic-level shifter
- Raspberry Pi 5

This folder is included because Stardance/Outpost examples sometimes use a `pcb/` folder. For Car Vault, the wiring diagram in `assets/wiring-diagram.png` and `assets/wiring-diagram.md` explains the wiring instead of a custom PCB design.
